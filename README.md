# Kinomix Button for Kinopoisk

[![Version](https://img.shields.io/badge/version-6.0-brightgreen)](https://github.com/accidettrauma/kinomix-kinopoisk-button)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Удобная **круглая кнопка** Kinomix прямо в интерфейсе Кинопоиска.

### ✨ Что делает скрипт
- Добавляет круглую серую кнопку 52×52 в правый блок кнопок
- Чёрная иконка по умолчанию
- При наведении иконка становится оранжевой, а фон кнопки немного темнеет
- Открывает фильм/сериал на **kinomix.web.app** в новой вкладке
- Работает на фильмах и сериалах
- Полностью вписывается в дизайн Кинопоиска

### 🚀 Установка

Нажми сюда → **[Установить скрипт](https://raw.githubusercontent.com/accidettrauma/kinomix-kinopoisk-button/main/kinomix-kinopoisk-button.user.js)**

Или установи вручную через Violentmonkey / Tampermonkey.
