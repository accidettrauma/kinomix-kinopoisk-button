# Kinomix Button for Kinopoisk

[![Version](https://img.shields.io/badge/version-1.0-brightgreen)](https://github.com/accidettrauma/kinomix-kinopoisk-button)

Удобная **кнопка**, которая добавляет прямую ссылку на просмотр фильма или сериала на **[Kinomix](https://kinomix.web.app)** прямо в интерфейс **[Кинопоиска](https://www.kinopoisk.ru)**.

### ✨ Что делает скрипт
- Добавляет круглую кнопку рядом с кнопкой "Буду смотреть"
- Открывает фильм/сериал на **kinomix.web.app** в новой вкладке

### 🚀 Установка

Установи **[Violentmonkey](https://violentmonkey.github.io)** / **[Tampermonkey](https://www.tampermonkey.net)**.

Нажми сюда → **[Установить скрипт](https://raw.githubusercontent.com/accidettrauma/kinomix-kinopoisk-button/main/kinomix-kinopoisk-button.user.js)**

Или добавь вручную
